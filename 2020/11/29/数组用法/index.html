    数组相关方法

    1. push

      • 功能： 在原数组末尾添加新元素
      • 参数： 非必须
      • 返回值： 数组长度
      • 是否改变原数组： 是

    2. unshift

      • 功能： 在原数组开始位置添加元素
      • 参数： 非必须
      • 返回值： 数组长度
      • 是否改变原数组： 是

    3. pop

      • 功能： 删除数组最后一位元素
      • 参数： 无
      • 返回值： 删除的元素
      • 是否改变原数组： 是

    4. shift

      • 功能： 删除数组第一位元素
      • 参数： 无
      • 返回值： 删除的元素
      • 是否改变原数组： 是

    5. indexOf

      • 功能： 返回某个指定的字符串值在字符串中首次出现的位置
      • XXX.indexOf(searchvalue,start)
      • 参数： searchvalue->必需:规定需检索的字符串值
      • 参数： start->非必需:规定检索的起始位置，不传则从首位开始
      • 返回值： 字符串中首次出现的下标，没有匹配则返回 -1
      • 是否改变原数组： 否

    6. reverse

      • 功能： 数组元素对调
      • 参数： 无
      • 返回值： 颠倒顺序后的数组
      • 是否改变原数组： 是

    7. concat

      • 功能： 数组合并任意值
      • 参数： 任意
      • 返回值： 合并后的数组
      • 是否改变原数组： 否

    8. slice

      • 释义：把…切成（薄）片

      • 功能： 对数组进行截取

      • 参数： slice(start,end)

      • 返回值： 截取后的数组（不包括end的元素）

      • 是否改变原数组： 否

    9. splice

      • 释义：连接；拼接
      • 功能： 该方法向或者从数组中添加或者删除项目
      • 参数： splice（index, Deletehowmany, newitemadded）
      • 返回值： 返回被删除的项目
      • 是否改变原数组： 是

    10. slice 与 splice 区别

      • 两者都有截取功能，且都返回被截取的结果并生成一个数组
      • splice，还有替换功能，即在截取的地方，添加需要添加的内容
      • slice不会改变原数组，splice会改变原数组
